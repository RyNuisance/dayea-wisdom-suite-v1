Recon — Network Scanner

Recon walks the perimeter of your network. It discovers every active device, checks which ports are open (like checking which doors and windows are unlocked), identifies what software is running, and attempts to detect the operating system.

What it finds: Live hosts, open ports (1–1024 by default), service banners, OS fingerprints, network topology.

Intel — Vulnerability Assessment

Intel takes what Recon found and goes deeper. It checks every service against known vulnerability databases (CVEs), tests for weak configurations, outdated software versions, and default credentials.

What it finds: Known CVEs, SSL/TLS weaknesses, default passwords, open admin panels, exposed sensitive files.
